Konfigurasi API Google Sheets

Cara Mengatasi Error 401 (Unauthorized):

  1. Buka Google Cloud Console
  2. Pilih project yang sama dengan API Key Anda
  3. Aktifkan Google Sheets API jika belum aktif
  4. Buat API Key baru atau gunakan yang sudah ada
  5. Bagikan spreadsheet Anda dengan izin "Anyone with the link can view"
  6. Masukkan API Key yang valid di atas dan klik Simpan
