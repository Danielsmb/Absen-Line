Metode 1: API Google Sheets

Masukkan API Key Google Sheets Anda untuk menghubungkan aplikasi dengan spreadsheet:

Cara Mendapatkan API Key:

  1. Buka Google Cloud Console
  2. Buat project baru atau pilih project yang sudah ada
  3. Aktifkan Google Sheets API
  4. Buat API Key di bagian "Credentials"
  5. Bagikan spreadsheet Anda dengan izin "Anyone with the link can view"
  6. Atur izin API Key untuk mengizinkan akses dari domain Anda
